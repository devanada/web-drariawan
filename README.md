# Dr. Ariawan Project

## Run Locally

Clone the project

```bash
  git clone git@github.com:dr-Ariawan-s-Project/web-drariawan.git
```

Go to the project directory

```bash
  cd web-drariawan
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
